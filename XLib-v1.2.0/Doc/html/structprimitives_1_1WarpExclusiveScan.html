Detailed Description

template<int WARP_SZ = 32>
struct primitives::WarpExclusiveScan< WARP_SZ >

Support structure for warp-level exclusive scan. 

Input:  1 2 3 4
Output: 0 1 3 6 (10)
Precondition
WARP_SZ must be a power of 2 in the range 1 ≤ WARP_SZ ≤ 32
Template Parameters
WARP_SZsplit the warp in WARP_SIZE / WARP_SZ groups and perform the exclusive prefix-scan in each groups. Require log2 ( WARP_SZ ) steps
